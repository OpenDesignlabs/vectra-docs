## What is Vectra?
Vectra is a SaaS visual website builder with a React/TypeScript frontend, a Rust WebAssembly engine for performance-critical operations, and Google WebContainers for a real in-browser Node.js VFS. It lets you design pages visually, preview them live, and export clean Next.js/Vite code.
