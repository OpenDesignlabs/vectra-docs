# Quickstart

There are four ways to start a new project in Vectra. Pick the one that fits how you work.

## Path 1 — Start from scratch

This is the fastest way to get a feel for Vectra.

**1. Open Vectra and create a new project.**
From the dashboard, click **New Project**. Choose a framework (Next.js or Vite) and give it a name.

**2. You'll land on the canvas.**
Your project starts with a single blank page called "Home" at the `/` route.

**3. Add your first element.**
Click the **+** button in the left panel (or press `A`) to open the Insert drawer. Click any element — try dragging a **Section** onto the canvas to start with a full-width block.

**4. Edit it.**
Click the element to select it. The right panel shows all its properties — size, color, padding, font. Change something.

**5. Add more pages.**
Click the **Pages** icon in the left panel. Hit **+ New Page**, give it a name like "About", and it gets a `/about` slug automatically.

**6. Export when ready.**
Click the **Export** button in the top bar. Download your project as a ZIP. Done.

<Callout type="tip">
  You don't need to install anything. Vectra runs entirely in the browser — open it and start building.
</Callout>

## Path 2 — Generate with AI

If you have an idea but don't want to drag and drop from zero, describe it to Vectra's AI and it will build a starting layout for you.

**1. Create a new project** (same as above).

**2. Open the AI panel.**
Look for the sparkle icon `✦` in the left toolbar, or press `Ctrl + K`.

**3. Describe what you want.**
Type something like:

> "A landing page for a mobile app with a hero section, three feature cards, and a pricing table."

**4. Vectra generates the layout.**
The AI places sections and elements on your canvas. It's not a finished design — it's a fast starting point you then edit visually.

**5. Edit and refine.**
Click any generated element and change it. Add your real copy, swap images, adjust colors from the Theme panel.

<Callout type="info">
  AI generation works best with clear, specific prompts. Mention the type of page, the sections you want, and any style preference (minimal, bold, dark, etc.).
</Callout>

## Path 3 — Import from Figma

If you have an existing Figma design, you can bring it into Vectra as an editable canvas.

**1. Open the Figma Import panel.**
Click the Figma icon in the left toolbar.

**2. Paste your Figma file URL.**
Make sure the file is set to "Anyone with the link can view" in Figma's share settings.

**3. Select which frame or page to import.**
Vectra fetches your design and converts it into editable canvas elements — real containers, text nodes, and images.

**4. Keep editing in Vectra.**
The imported design is now fully editable. Rearrange elements, change styles, add new sections on top of the import.

<Callout type="warn">
  Figma import converts static designs into web layouts. Complex Figma components with variants or auto-layout may need some manual adjustment after import.
</Callout>

## Path 4 — Browse the Marketplace

If you want professionally designed headers, heroes, and footers, start with the Marketplace and pull in high-fidelity modules.

**1. Open the Marketplace.**
Click the Shopping Bag icon `🛍️` in the left sidebar to open the Marketplace.

**2. Explore categories.**
Browse through specific UI sections like **Heros**, **Navigation**, **Pricing**, and **Features**. Everything is built using pure React and Tailwind CSS.

**3. Interactive Previews.**
Hover over any component to see it in action. Use the **Live Preview** to test its responsiveness across different breakpoints.

**4. Drop into your project.**
Click the component's **Insert** button to instantly add it to your current active canvas.

**5. Customize natively.**
After inserting from the Marketplace, the component is yours. Every property, class, and logic state is editable right in the Studio.

<Callout type="tip">
  The Marketplace components are version-controlled. If the original author updates their component, you'll see a small notification in your Layers panel.
</Callout>
