# Keyboard Shortcuts

A full list of keyboard shortcuts to speed up your workflow in Vectra.

## Canvas

| Action | Windows / Linux | Mac |
|---|---|---|
| Undo | `Ctrl + Z` | `Cmd + Z` |
| Redo | `Ctrl + Shift + Z` | `Cmd + Shift + Z` |
| Select all | `Ctrl + A` | `Cmd + A` |
| Duplicate element | `Ctrl + D` | `Cmd + D` |
| Delete element | `Delete` or `Backspace` | `Delete` or `Backspace` |
| Copy | `Ctrl + C` | `Cmd + C` |
| Paste | `Ctrl + V` | `Cmd + V` |
| Deselect | `Escape` | `Escape` |

## Zoom & navigation

| Action | Windows / Linux | Mac |
|---|---|---|
| Zoom in | `Ctrl + +` | `Cmd + +` |
| Zoom out | `Ctrl + -` | `Cmd + -` |
| Fit page to view | `Ctrl + 0` | `Cmd + 0` |
| Zoom to 100% | `Ctrl + 1` | `Cmd + 1` |
| Pan canvas | `Space + drag` | `Space + drag` |

## Tools

| Action | Shortcut |
|---|---|
| Open Insert drawer | `A` |
| Open AI panel | `Ctrl + K` |
| Select tool | `V` |
| Hand tool (pan) | `H` |
| Text tool | `T` |

## Editing

| Action | Windows / Linux | Mac |
|---|---|---|
| Edit text (enter text mode) | `Enter` or double-click | `Return` or double-click |
| Exit text mode | `Escape` | `Escape` |
| Move element 1px | Arrow keys | Arrow keys |
| Move element 10px | `Shift + Arrow` | `Shift + Arrow` |
