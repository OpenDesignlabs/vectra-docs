## Write mutation flow
All Write tools return a `__vectra_mutation__` payload over the SSE stream. `MCPPanel` listens to the SSE and dispatches a `vectra:mcp-command` CustomEvent — which `ProjectContext` catches and applies to React state. VFS is never written by the MCP server directly.

<CodeBlock lang="json">
{`{
  "__vectra_mutation__": true,
  "op": "ADD_ELEMENT",
  "newId": "tex-mcp1a2b3c",
  "parentId": "page-home",
  "element": { /* VectraNode */ }
}`}
</CodeBlock>
