# Advanced AI Generation (Stitch)

Vectra's AI engine, known as **Stitch**, goes far beyond simple layout generation. It creates production-ready React components, imports entire projects from ZIP files, and can even crawl a live URL to bring it onto your canvas.

## 1. Feature Generation (Prompt-to-Component)

Instead of just layout blocks, Stitch generates **real React functional components** styled with Tailwind CSS.

- **How to use:** Open the AI panel (`✦` or `Ctrl + K`) and ensure the **AI** tab is selected.
- **Why it's better:** The AI writes idiomatic code. If you ask for a "Modern SaaS Pricing Table", it doesn't just place containers—it builds a complete, interactive component that you can instantly edit.
- **Direct Insert:** Once generated, click **Insert to Canvas** to place the component on your active page.

## 2. Project Import (Zip-to-Vectra)

Have an existing React or Next.js project you want to move into Vectra?

- **How to use:** Go to the AI panel and select the **Zip** tab. Drop your `.zip` archive.
- **Workflow:** Stitch parses your project's file structure, identifies components, and maps them to **VectraNodes**. 
- **Result:** Your entire local project becomes an editable Vectra canvas automatically.

## 3. URL Crawler (Site-to-Canvas)

Want to use an existing website as a starting point for a new project?

- **How to use:** In the AI panel, select the **URL** tab and paste a link (e.g., `https://your-competitor.com`).
- **Crawler Logic:** Stitch fetches the HTML/CSS of the target site and intelligently reconstructs the layout using Vectra's native components.
- **Editable State:** The imported site isn't just a static image—it's a fully editable node tree where you can swap colors, text, and layout on the fly.

<Callout type="info">
  **Note on Quality:** AI-generated components and imports are high-fidelity drafts. You should always refine the generated copy, swap placeholder images, and ensure the final theme matches your brand in the **Theme Panel**.
</Callout>
